# Uber-Trip-Analysis and Prediction

## Project Overview
This project analyzes Uber trip data and predicts hourly trip demand using Machine Learning models.

## Dataset
Uber trip dataset (April–September 2014)

/kaggle/input/datasets/ganusha/dataset-jan-feb-foil

/kaggle/input/datasets/ganusha/dataset-april-sep

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- XGBoost

## Machine Learning Models
- Random Forest Regressor
- Gradient Boosting Regressor
- XGBoost Regressor
- Ensemble Model

## Evaluation Metric
Mean Absolute Percentage Error (MAPE)

## Results
XGBoost achieved the best performance with the lowest prediction error.

## Key Features
- Time feature engineering
- Trip demand visualization
- Model comparison
- Prediction vs actual analysis

## Conclusion
The models effectively capture temporal patterns in Uber trip demand and can help forecast ride demand for better resource planning.
